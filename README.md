# Customer-Feedback-Analysis

## Project Overview

Customer Feedback Analysis is a data analytics and Natural Language Processing (NLP) project developed using Python and Power BI. The project focuses on analyzing Amazon customer reviews to identify customer sentiments, product feedback trends, and customer satisfaction patterns.

The project uses sentiment analysis, word cloud visualization, topic modeling, and interactive dashboard visualization to generate meaningful insights from customer review data.

---

# Features

* Customer Sentiment Analysis
* Data Cleaning and Preprocessing
* Word Cloud Visualization
* Topic Modeling using LDA
* Rating Distribution Analysis
* Category-wise Product Analysis
* Interactive Power BI Dashboard
* Customer Feedback Insights

# Dataset Information

The dataset used in this project contains Amazon customer review and product-related information such as:

* Product Name
* Product Category
* Ratings
* Discounted Price
* Review Title
* Review Content
* Rating Count
* Product Description

The dataset was collected from Kaggle and used for sentiment analysis and visualization.
